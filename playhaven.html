Hello PlayHaven!

This project will be an on-going project that will be pushed up to GitHub.

The script will make a AJAX call to Itunes on keystroke up anytime there is a .5 second delay between characters. The content field will update as needed as the user is typing in the search field. This is can be observed by typing in the search box very slowly.

I will be updating the code base with Backbone.js and Handlebar.js for my templating needs starting next week.

As it stands right now, the total time it took to complete the code base stands at 3 hours. This was partially due to several bugs that I ran into while utilizing Itunes API.
